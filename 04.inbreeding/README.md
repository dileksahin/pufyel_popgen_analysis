## 04.inbreeding
In this pipeline, per individual inbreeding coefficient was calculated using ngsF V 1.2.0 (Vieira et al., 2013) and used as priors in SFS estimation.

### Reference genome
Balearic Shearwater (*Puffinus mauritanicus*) genome
Cuevas-Caballé et al. (2021)
BioProject: PRJNA780920
BioSamples: SAMN24039388, SAMN23492024, SAMN23212142

### Software versions
- ANGSD V 0.940(Korneliussen et al., 2014)
- ngsF V 1.2.0 (Vieira et al., 2013)

### Usage
1. Edit config.sh to set paths for your system
2. Submit: bash run_inbreeding_pipeline.sh
